# Motorized Couch

## Installation

Install PyVESC by typing:
```bash
pip install git+https://github.com/LiamBindle/PyVESC.git
```

And also install the `requirements.txt` with `pip3 install -r requirements.txt`

To use, run `joystick_esc.py`
